Sargam colors are the language of the beauty, the warmth, and the caress, that comes from the heart of the world. The most direct expression of beauty, with a form and spirit which is one and simple, and least encumbered with anything extraneous.

A system of emancipation which makes it easy to recognize the beauty in small things. True makers, they who are seers, seek to express the universe with sargam colors.

Installation

Install Sargam Colors from your terminal via npm. Current version is 1.2.2

        npm i @sargamdesign/colors

Import vanilla CSS.

        import "@sargamdesign/colors/dist/themes/light.css";
        import "@sargamdesign/colors/dist/themes/dark.css";

To get started quickly, you can use the CDN files. However, using the CDN approach is not recommended for production. It's intended for prototyping only.

        <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@sargamdesign/colors@1.2.1/dist/themes/light.min.css">
        <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@sargamdesign/colors@1.2.1/dist/themes/dark.min.css">

Usage

Sargam colors provide the light and dark themes bundled as raw CSS files. Below you can find usage example:

        .button { background-color: var(--button-primary);
                  color: var(--text-on-color); }
        .button:hover { background-color: var(--button-primary-hover); }
        .button:active { background-color: var(--button-primary-active); }
